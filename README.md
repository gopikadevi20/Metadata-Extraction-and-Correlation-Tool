Metadata-Extraction-and-Correlation-Tool
Project Description
The Metadata Extraction and Correlation Tool is a digital forensic application that extracts metadata from files, verifies file integrity, detects tampering, correlates metadata across files, and generates automated forensic reports using a user-friendly GUI.

Key Features
Extracts metadata from images, documents, and media files
Verifies file integrity using hash algorithms (MD5, SHA-256)
Validates file signatures to detect tampering
Correlates metadata across multiple files
Detects metadata anomalies
Maintains logs and digital evidence tracking
Generates automated CSV/PDF forensic reports
Supports batch file and folder processing

System Architecture
GUI Layer
Metadata Processing Engine
Integrity & Signature Validation Module
Correlation & Anomaly Detection Engine
Database & Logging Module
Report Generation Module
